# 读书笔记

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把你读到的内容粘贴进去。

## 提示词

```
请帮我整理一份读书笔记。

书名：「」
作者：「」
我摘录/想整理的内容：
「粘贴你的摘抄、划线、零散想法」

请按以下结构整理：

## 核心观点（3-5 个）
作者最想传达的关键思想，每个用 1-2 句话概括

## 金句摘录（5-10 条）
最有启发的原文句子，标注页码（如果有）

## 我的启发
基于你的摘录，提炼出：
- 这本书改变了我的什么认知？
- 有哪些可以马上用到生活中的？

## 行动清单
列出 3 件看完这本书后可以立刻做的事

要求：
- 用自己的话总结，不要照抄原书
- 非虚构类突出"可执行性"，虚构类突出"情感共鸣点"
- 如果摘录内容太少，请提示我补充
```

## 示例

略（根据你输入的书和内容生成）

## 技巧
- 读完一章就记一页，别攒到全书读完再整理
- 好的读书笔记是"看完一年后翻出来还能用"，不是"看完觉得写得好"
- 一本书的笔记里如果没有"我能做什么"，这本书白读了
