# 翻译助手

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
把你想要翻译的文字粘贴到 AI 对话框，加上以下提示词。

## 中译英

```
请将以下中文翻译成英文。

要求：
- 翻译自然流畅，符合英语母语者的表达习惯
- 不要逐字翻译，要意译
- 保持原文的语气和风格
- 专业术语翻译准确

中文原文：
「粘贴要翻译的内容」
```

## 英译中

```
请将以下英文翻译成中文。

要求：
- 翻译自然流畅，符合中文表达习惯
- 不要翻译腔（如避免过多使用"的""地""被"字句）
- 长句适当拆分为短句
- 专业术语翻译准确

英文原文：
「粘贴要翻译的内容」
```

## 学术翻译（论文摘要等）

```
请将以下中文摘要翻译成地道的学术英文。

要求：
- 使用学术写作风格
- 被动语态优先
- 术语保持一致性
- 符合 SCI/SSCI 期刊的英文摘要规范

中文原文：
「粘贴你的中文摘要」
```

## 技巧
- 翻译整篇文章时，先让 AI 翻译一遍，再让它"润色"一遍
- 专业文档翻完后，告诉 AI "请检查是否有术语翻译不一致的地方"
- AI 翻译最大的问题是"太直译"——加一句"用地道的中文/英文重新表达"
