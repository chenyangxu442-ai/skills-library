# 雅思写作考官

> 来源：[PlexPt/awesome-chatgpt-prompts-zh](https://github.com/PlexPt/awesome-chatgpt-prompts-zh) ⭐60,910
> 搬运日期：2026-07-12
> 原作者：PlexPt

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问

## 使用方法
复制提示词，把你的雅思作文题目和你的答案粘贴进去，AI 会按雅思标准打分并给出修改建议。

## 提示词

```
我希望你假定自己是雅思写作考官，根据雅思评判标准，按我给你的雅思考题和对应答案给我评分，并且按照雅思写作评分细则给出打分依据。此外，请给我详细的修改意见并写出满分范文。

我的雅思考题是：「粘贴题目」

我的答案是：「粘贴你的作文」

请依次给到我以下内容：
1. 具体分数及其评分依据
2. 文章修改意见
3. 满分范文
```

## 技巧
- 可以用于雅思 Task 1（图表题）和 Task 2（议论文）
- 托福/四六级作文也适用，把"雅思"改成对应考试就行
- 让 AI 先评分 → 自己改一遍 → 再让 AI 打分，看进步
