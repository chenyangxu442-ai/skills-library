# 知识讲解（费曼学习法）

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，把你想学的概念告诉 AI，它会用最简单的方式给你讲明白。

## 提示词

```
请使用费曼学习法给我讲解一个概念。

我想了解的概念：「如：什么是区块链 / 相对论 / 神经网络」
我的知识背景：「如：完全没有基础 / 学过高中物理 / 会编程但数学一般」

要求：
1. 用最简单的语言解释，假设我是一个完全不懂的人
2. 用生活中的例子类比（不要说"就像……"，要说"好比你在菜市场买菜……"）
3. 先讲"是什么"，再讲"为什么重要"，最后讲"怎么用"
4. 讲完后，用 3 个问题测试我是否真的理解了
5. 如果有常见的误解，提醒我
6. 如果概念本身就有争议（比如经济学流派），告诉我争议在哪
```

## 进阶版：苏格拉底式提问

```
不要直接给我解释「概念」。请用苏格拉底式提问法：
1. 先问我"你觉得「这个概念」是什么？"
2. 根据我的回答，指出我的理解缺口
3. 通过追问引导我自己想明白
4. 实在卡住了再给提示
5. 最后总结正确的理解

开始吧！
```

## 技巧
- 如果你解释不清楚一个概念，说明你还没真懂——这是个自检工具
- 学习新概念后，试着用自己的话复述给 AI 听，让它指出你哪里理解错了
- 让 AI 给你出 3 道练习题，比听讲 10 遍都管用
