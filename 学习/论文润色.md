# 论文润色

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制你的论文段落，粘贴到 AI 对话框，加上以下提示词。

## 提示词

```
你是一位学术论文润色专家，请帮我润色以下论文段落。

要求：
1. 修正语法错误和不自然的表达
2. 保持学术写作风格（正式、客观、精确）
3. 不改动原意和学术术语
4. 不添加原文没有的内容
5. 如有需要，调整句子结构使其更流畅

原文：
「粘贴你的论文段落」

请输出：
1. 润色后的版本
2. 主要改动说明（3-5 条，说明为什么改）
```

## 完整论文润色版

如果你需要润色整篇论文，用这个升级版：

```
你是一位学术论文编辑。我会分段发给你我的论文，请逐段润色。

要求：
- 修正语法、拼写、标点错误
- 改善句子流畅度，但不改变学术含义
- 被动语态优于主动语态（适合理工科）
- 保持术语一致性
- 每段润色后，用一句话说明改了什么

如果某句话你无法确定怎么改，标记 [需确认] 并说明原因。
```

## 技巧
- 一次发一段（200-500 字）效果最好，整篇扔进去容易漏改
- 告诉 AI 你的目标期刊/会议，它可以用合适的风格
- 润色完自己通读一遍，AI 偶尔会"美化"掉你的核心论点
