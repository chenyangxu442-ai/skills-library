# 工作邮件

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位专业的商务邮件撰写专家。请帮我写一封工作邮件。

邮件类型：「如：跨部门协作请求 / 项目进度同步 / 会议邀请 / 客户跟进 / 投诉回复」
收件人身份：「如：市场部总监 / 客户公司负责人」
我想传达的核心信息：
「用大白话写你想说什么，比如：我想让他们在周五之前把素材给我，不然下周上线来不及」

要求：
- 标题清晰，一眼看出什么事
- 开头一句话说清来意（不要"见信好""祝你工作顺利"之类的寒暄堆砌）
- 正文结构：背景(1句) → 需求/信息 → 时间节点 → 行动要求
- 语气根据邮件类型调整（跨部门用协商语气，客户用专业且温暖的语气）
- 结尾明确 Call to Action（对方需要做什么，什么时候前做完）
- 如果事情紧急，说明"为什么紧急"
- 控制在 200 字以内
```

## 示例

**输入：** 催设计部出活动 banner，周三之前要，不然来不及测试上线

**输出：**

**主题：【需配合】618活动banner素材需求，请周三18:00前提供**

Hi 王姐，

618大促活动预计下周一上线，需要我们提前完成前端开发和测试。

烦请设计部在**本周三（7月15日）18:00前**提供以下素材：
1. 首页轮播 banner × 3（尺寸 1920×600）
2. 活动专题页头图 × 1

如有困难请随时沟通，我这边可以协调排期。感谢支持！

张明
产品部

## 技巧
- 正文 200 字以内的邮件，回复率最高
- 时间节点用加粗，让人一眼看到
- 跨部门邮件抄送对方领导效果好（但要谨慎用）
