# 辞职信

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
请帮我写一封辞职信。

我的情况：
- 岗位：「如：软件开发工程师」
- 公司/团队：「如：XX公司技术部」
- 离职原因简述：「如：个人职业规划调整」
- 对公司的态度：「如：感谢公司两年来的培养」
- 最后工作日：「如：2026年8月15日」
- 是否愿意配合交接：「如：会完整交接所有工作」

要求：
- 语气得体外加，不要有任何抱怨
- 表达感谢但不谄媚
- 明确最后工作日
- 承诺配合交接
- 控制在 200 字以内
- 好聚好散，留个后路
```

## 示例

**主题：辞职申请**

尊敬的主管：

我在公司工作的两年里，收获了很多成长和经验，由衷感谢您和团队的信任与支持。

经过慎重考虑，我决定因个人职业规划原因辞去软件开发工程师一职。我的最后工作日为 2026 年 8 月 15 日。在此之前，我会完成所有手头工作的交接，确保团队不受影响。

感谢公司提供的平台和机会，祝愿公司业务蒸蒸日上。

此致
敬礼

张三
2026年7月12日

## 技巧
- 辞职信不要写真实原因（尤其是负面原因），面谈时再说
- 提前跟直属领导口头沟通后再发正式邮件
- 最后工作日按合同约定给足通知期（通常 30 天）
