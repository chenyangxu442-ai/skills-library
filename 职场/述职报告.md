# 述职报告

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位资深的职场汇报专家。请帮我写一份述职报告（或晋升答辩材料）。

我的岗位：「如：后端开发工程师 / 产品经理 / 运营主管」
述职周期：「如：2026年上半年 / 入职一年」
我的核心工作：「简要列出你的主要工作，流水账即可」

请按以下结构生成述职报告：

## 一、工作概述
用 3 句话概括这段时间的核心贡献

## 二、重点项目成果
选择 2-3 个最重要项目，每个按 STAR 法则展开：
- 背景（Situation）：为什么要做
- 任务（Task）：我要做什么
- 行动（Action）：我具体做了什么
- 结果（Result）：带来了什么变化（必须带数据）

## 三、数据亮点
用 3-5 个关键数据展示工作价值

## 四、成长与反思
- 这段期间最大的成长是什么
- 遇到过什么困难，怎么解决的

## 五、下阶段规划
列出下阶段的 3 个重点方向

要求：
- 用数据和事实说话，避免空泛
- STAR 部分每个项目 200-300 字
- 整体语气自信但不傲慢
- 突出"我"的贡献，不是"团队"的贡献
```

## 示例

略（每个岗位差异太大，AI 会根据你的输入生成个性化内容）

## 技巧
- STAR 法则最关键的是 Result——必须说清楚"因为我做了什么，所以结果变了什么"
- 提前收集好数据：用户增长、收入变化、效率提升、bug 减少……
- 晋升答辩的话，加一句"我做了什么别人做不到的"
