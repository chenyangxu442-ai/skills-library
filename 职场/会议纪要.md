# 会议纪要

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位专业的会议记录专家。请将我的会议记录整理成一份标准会议纪要。

会议主题：「」
会议时间：「」 
参会人员：「」
我的会议记录（速记/流水账）：
「粘贴你的会议笔记，越乱越好」

请按以下结构整理：

## 会议信息
- 主题 / 时间 / 地点 / 参会人 / 主持人

## 核心结论
列出本次会议达成的 3-5 个关键结论，用加粗标出

## 讨论要点
按议题分组，每个议题：
- 背景简述（1 句话）
- 各方观点（谁说了什么）
- 最终决定

## 待办事项（最重要！）
用表格列出：
| 事项 | 负责人 | 截止时间 | 优先级 |
|------|--------|----------|--------|

## 下次会议
- 时间 / 议题预告

要求：
- 待办事项必须具体可执行，不说"跟进XX"，说"在周三前完成XX并发邮件给YY"
- 每条待办有且只有一个负责人
- 删除闲聊和无关内容
```

## 示例

略（根据你的会议内容自动生成）

## 技巧
- 开会时记关键词就够了，会后把关键词扔给 AI 整理
- 待办事项是会议纪要的灵魂——没待办的会等于白开
- 发出去之前确认每个负责人看到了自己的待办
