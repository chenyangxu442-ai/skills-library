# 周报生成器

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位专业的周报撰写专家。请根据我本周的工作内容，生成一份结构清晰的周报。

我的工作内容如下：
「粘贴你这周干了什么，流水账就行」

请按以下结构生成周报：

## 本周核心产出
- 列出 3-5 个最重要的成果，每个用一句话概括

## 关键进展
- 每个成果展开 2-3 句，说明具体做了什么、有什么数据变化

## 下周计划
- 列出下周 3 个优先级最高的事情

## 需要协助
- 有什么卡点需要领导或同事支持的

要求：
- 语言简洁专业，不啰嗦
- 尽量带数据（完成率、增长百分比、时间节点）
- 总字数控制在 300-500 字
- 不要流水账，要突出结果和影响
```

## 示例

**输入：**
本周做了用户模块的接口开发，修了几个 bug，开了两次评审会，帮前端联调了三天。

**输出：**
```
## 本周核心产出
1. 完成用户模块 6 个核心接口开发，通过联调
2. 修复线上 bug 4 个，无回滚事故
3. 参与 2 次技术评审，输出接口文档一份

## 关键进展
- 用户模块接口开发完成率 100%，前端联调通过，已提测
- 修复用户登录超时、头像上传失败等 4 个 bug，影响用户数约 200 人
- 完成接口文档，同步给前端和测试团队

## 下周计划
1. 配合测试完成用户模块 bug 修复
2. 开始订单模块接口设计
3. Code Review 新人代码

## 需要协助
前端排期较满，下周联调可能需要协调人力
```

## 技巧
- 日常随手记一下做了什么，别等到周五才回忆
- 数据比形容词有说服力：不说"提升了很多"，说"提升 40%"
- 周报是写给你未来的自己升职加薪用的，不只是给老板看的
