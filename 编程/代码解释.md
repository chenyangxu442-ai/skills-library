# 代码解释

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制你要理解的代码，粘贴到 AI 对话框，加上以下提示词。

## 提示词

```
请解释以下代码。我是一名「如：初学者 / 有其他语言经验的开发者」，「熟悉/不熟悉」「如：Python」。

要求：
1. 先一句话概括这段代码在做什么
2. 逐行/逐块解释关键逻辑
3. 指出代码中用到的核心技术点
4. 如果有更好的写法，给出改进建议
5. 如果有潜在的 bug 或性能问题，指出来

代码：
「粘贴你的代码」
```

## 进阶版：逐行注释

```
请给以下代码加上详细的中文注释，每一行都解释。

代码：
「粘贴你的代码」
```

## Debug 版

```
以下代码运行时出现了这个错误：「粘贴错误信息」
请帮我分析原因并给出修复方案。

代码：
「粘贴你的代码」
```

## 技巧
- Claude 在代码理解方面表现最好，复杂的逻辑用它
- 一次贴一段代码（50-200 行），贴整个项目 AI 会漏掉细节
- 问"这段代码有没有安全问题"是一个好习惯
