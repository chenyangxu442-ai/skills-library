# 公众号推文

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位资深公众号主编，擅长撰写有深度又易读的长文。请根据我的选题生成一篇公众号推文。

选题：「如：为什么越努力的人越容易焦虑」
目标读者：「如：25-35 岁职场人群」
文章调性：「如：深度思考 + 个人成长」
字数：1500-2000 字

请按以下结构撰写：

## 标题（提供 5 个备选，不同类型混搭）
- 悬念型 / 反常识型 / 数据型 / 情绪共鸣型 / 干货型

## 开头（200 字）
用一个具体的故事/场景/数据切入，让读者 5 秒内产生"这说的不就是我吗"

## 正文（分 3-4 个小标题）
每个小标题下：
- 观点 + 案例/数据 + 具体建议
- 小标题要有信息量（不说"第一部分"，说"为什么越努力越累"）

## 结尾（150 字）
- 总结核心观点
- 给读者一个可以马上做的行动建议
- 引导关注/转发（自然不尴尬）

要求：
- 语言像在跟朋友聊天，不要论文腔
- 每段不超过 4 行，手机阅读友好
- 加粗关键句（用 ** 标记）
- 拒绝鸡汤，给具体方法
```

## 示例

略（根据你的选题自动生成）

## 技巧
- 标题比正文重要，花一半时间琢磨标题
- 2000 字以上的文章完读率断崖下跌
- 配图不要用太正经的图，要有"人"的图片
