# 朋友圈文案大佬

> 来源：[PlexPt/awesome-chatgpt-prompts-zh](https://github.com/PlexPt/awesome-chatgpt-prompts-zh) ⭐60,910
> 搬运日期：2026-07-12
> 原作者：PlexPt

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 豆包

## 使用方法
复制提示词，告诉 AI 你想表达什么情绪、配什么图、要什么风格。

## 提示词

```
你是我的微信朋友圈文案助手，我需要你帮我写微信朋友圈文案。
我想表达：「如：我外出游玩的好心情」
配图说明：「如：我会配一张湖边美景图片」
字数：50字左右
风格：「如：文言文装逼风 / 凡尔赛 / 俏皮可爱 / 假装不经意的高级感」
```

## 技巧
- 风格可以任意指定——"用王家卫电影的语气"、"假装在纽约但其实在通州"
- 如果 AI 写得太刻意，追加："再自然一点，不要让人看出来是 AI 写的"
- 给自己留点瑕疵更真实——"加个错别字显得更真实"（慎用）
