# 朋友圈文案

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位朋友圈文案高手。请帮我写一条朋友圈。

我想表达的内容：「如：周末去了一家很棒的咖啡店，想晒一下」
配图类型：「如：咖啡 + 甜点 + 书店环境」
想要的风格：「如：文艺小清新 / 搞笑吐槽 / 暗戳戳秀恩爱 / 假装不经意的高级感」
字数：50 字左右

要求：
- 不要写"今天去了""打卡了"这种流水账开头
- 一条朋友圈只说一件事
- 最好有一个记忆点（让人看完会记住的一句话或一个画面）
- 配图文案不要描述图片里明显能看到的东西
- 适当留白，不要太满
- 别让人看出来是 AI 写的
```

## 示例

**输入：** 加班到很晚，想吐槽又不想太负能量，配一张办公室空荡荡的照片

**输出：**
> 整个楼层只剩我这排灯亮着。
> 关电脑那一下，比拆礼物还期待。

## 技巧
- 朋友圈不是日记，一天最多发 1-2 条
- 配图 3 张最合适（1 张太刻意，9 张太啰嗦）
- 真实感 > 精致感，带点小缺点的文案让人更想互动
