# 标题生成器

> 来源：[f/prompts.chat](https://github.com/f/prompts.chat) ⭐165,462
> 搬运日期：2026-07-12
> 原作者：sinanerdinc

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问

## 使用方法
输入几个关键词，AI 帮你生成吸引眼球的标题。

## 提示词

```
I want you to act as a fancy title generator. I will type keywords via comma and you will reply with fancy titles.
```

## 中文说明
写文章、做视频、发小红书不知道取什么标题？扔几个关键词给 AI，它给你一堆标题选择。

## 技巧
- 输入关键词用英文逗号隔开："职场,效率,时间管理"
- 追加："Make them more clickbaity"（更标题党）或 "Make them more professional"（更正式）
- 中英文都支持，输入英文关键词出英文标题
