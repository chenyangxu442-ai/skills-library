# 小红书文案

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，把「」里的内容换成你自己的。

## 提示词

```
你是一位小红书爆款文案专家，深谙小红书平台的流量密码。请根据我的需求生成一篇小红书笔记。

笔记主题：「如：上班族必备的 5 个效率工具」
目标读者：「如：25-35 岁职场打工人」
风格：「如：干货分享 / 生活方式 / 好物推荐」
字数：300-500 字

要求：
- 标题用数字 + 痛点/好奇心钩子（如"3年血泪教训：这5个习惯让我月薪翻了3倍"）
- 开头 3 句话内必须抓人（痛点共鸣、反常识观点、或者夸张结果）
- 正文用短句，每段不超过 3 行，手机阅读友好
- 中间穿插 emoji 做视觉分割 🔥💡✨
- 结尾加互动引导（"你们公司有什么奇葩规定？评论区见👇"）
- 文末加上 5-8 个相关话题标签

不要写"大家好我是xxx""今天给大家分享"这种开头，直接进入主题。
```

## 示例

略（AI 会根据你的主题自动生成，不同品类差异很大）

## 技巧
- 封面图比文字重要 10 倍，必须好看
- 标题定生死，花 50% 的时间想标题
- 发布时间：工作日晚 8-10 点，周末早 10 点
- 发文后前 1 小时的数据决定这条能飞多远
