# 旅行计划

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，填好你的旅行信息。

## 提示词

```
你是一位资深旅行规划师。请帮我制定一份旅行计划。

目的地：「如：成都」
出行时间：「如：2026年8月，5天4晚」
出行人数：「如：2人，情侣/亲子/朋友」
预算：「如：人均3000元（含机票住宿）」
兴趣偏好：「如：美食为主 / 自然风光 / 人文历史 / 网红打卡」
住宿要求：「如：干净就行 / 要有特色民宿 / 五星酒店」
交通方式：「如：公共交通 / 自驾 / 打车」

请按以下结构输出：

## 一、总体概览
- 最佳出行时间 / 天气提醒
- 总预算估算表（分项）

## 二、每日行程（按天展开）
```
Day 1：
  上午（9:00-12:00）：「景点/活动 + 交通方式 + 预计时间」
  中午（12:00-14:00）：「餐厅推荐（2-3个选项）」
  下午（14:00-18:00）：...
  晚上（18:00- ）：...
  住宿：...
```

## 三、美食清单
- 必吃榜 TOP 5
- 避雷：哪些网红店不值得排队

## 四、注意事项
- 需要提前预约的景点
- 当地交通 tips
- 安全提醒

要求：
- 行程不要太赶，留出弹性时间
- 每个景点之间考虑交通时间
- 餐厅推荐要有备选，以防排队太久
```

## 技巧
- 先让 AI 出总体方案，确认方向了再让它细化每天的行程
- AI 给的营业时间/价格可能不准，重要信息自己去官网确认
- 让 AI 帮你写一份"行李清单"，出发前照着检查
