# AI 取名

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，粘贴到 AI 对话框，填好你的需求。

## 提示词

```
你是一位专业取名顾问，精通中文姓名学、音韵学和传统文化。请帮我取名字。

取名类型：「如：宝宝取名 / 公司取名 / 品牌取名 / 笔名」
姓氏：「如：姓王」（如果是公司/品牌可跳过）
性别：「如：男 / 女 / 不限」
特殊要求：「如：最好带三点水 / 需要五行补金 / 要大气好记 / 参考古诗词」
风格偏好：「如：文雅 / 现代 / 霸气 / 可爱」

请提供 20 个名字，每个名字包含：
1. 名字
2. 寓意解释（简洁）
3. 出处（如有典故）

并标注你推荐的 TOP 3 以及推荐理由。

注意：
- 避开过于生僻的字（不会被电脑系统识别）
- 注意谐音（不能有不雅或负面的谐音）
- 读音要响亮（避免三个字同一个声调）
```

## 宝宝取名补充

如果是取宝宝名，可以加上：
```
补充信息：
- 出生年月：「2026年7月」
- 生肖：「马」
- 五行需求：「缺木」（如有找人算过）
- 避开字：「伟、强、红」（家族要避开的字）
- 兄弟姐妹名字：「哥哥叫王浩然」（希望名字风格协调）
```

## 技巧
- AI 可能不精确计算笔画/五行，涉及八字五行建议找人算好，让 AI 只做文字创意
- 取名后去公安局网站查重名率
- 多读几遍，让家人朋友也读读——不同人听到的谐音不一样
