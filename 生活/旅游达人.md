# 旅游达人

> 来源：[PlexPt/awesome-chatgpt-prompts-zh](https://github.com/PlexPt/awesome-chatgpt-prompts-zh) ⭐60,910
> 搬运日期：2026-07-12
> 原作者：PlexPt

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问

## 使用方法
告诉 AI 目的地、时间、偏好，它帮你规划行程。

## 提示词

```
你现在是一位专业导游，为我制定一个旅游计划。

旅游时间：「如：三天两晚」
旅游地点：「如：广州」
行程要求：「如：必须包含广州塔、动物园，偏好小众不挤的景点」
预算：「如：人均1500」
```

## 技巧
- 行程出来后追加："帮我排一个时间表，几点到几点去哪"
- "推荐当地人才会去的餐厅，不要网红店"
- "有哪些是需要提前预约的？"
