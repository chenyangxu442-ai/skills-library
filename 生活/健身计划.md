# 健身计划

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，告诉 AI 你的身体情况和目标。

## 提示词

```
你是一位专业健身教练。请根据我的情况制定一份个性化的健身计划。

我的情况：
- 年龄/性别：「如：28岁 男」
- 身高体重：「如：175cm / 80kg」
- 目标：「如：减脂到 70kg / 增肌 / 保持健康 / 练出腹肌」
- 运动经验：「如：完全没有 / 偶尔跑步 / 健身半年」
- 可用时间：「如：每周 3 天，每次 1 小时」
- 场地器材：「如：只有哑铃 / 健身房全器械 / 居家自重」
- 伤病/限制：「如：膝盖不好 / 腰不能弯」

请提供：

## 一、每周训练安排
```
周一：力量训练（胸+三头） 45分钟
周二：休息
...
```

## 二、每次训练的具体动作
每个动作列出：
- 动作名称
- 组数 × 次数
- 简单的动作要点（不要写"核心收紧"这种废话，说具体的）
- 常见错误（新手最容易做错的地方）

## 三、饮食建议（如果目标是减脂/增肌）
- 大概每天吃多少（不用精确到克）
- 什么可以吃 / 什么尽量少吃

## 四、预期效果
- 1个月 / 3个月 / 6个月 大概能看到什么变化

重要提醒：
- 给出的是适合新手的低门槛方案
- 任何让你关节痛（不是肌肉酸痛）的动作立刻停止
```

## 技巧
- AI 给的训练计划仅供参考，最好找真人教练纠正动作
- 饮食控制占减脂效果的 70%，练得再狠管不住嘴没用
- 坚持比强度重要——一周 3 次中等强度 > 一周 6 次练到趴
