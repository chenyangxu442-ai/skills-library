# 送礼建议

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，告诉 AI 你要送礼的对象和预算。

## 提示词

```
你是一位送礼顾问，擅长根据不同的人和场合推荐礼物。请帮我出主意。

送礼对象：「如：女朋友 / 老爸 / 客户 / 导师 / 闺蜜生日」
对方特点：「如：喜欢看书，文艺风格，30岁，不化妆」
送礼场合：「如：生日 / 结婚 / 过年 / 拜访感谢 / 道歉」
预算：「如：300-500元」
避雷：「如：不要香水 / 不要吃的」

请推荐 10 个礼物选项，每个包含：
1. 礼物名称 + 大概价格
2. 为什么适合这个人（结合对方特点说明）
3. 在哪里买（线上/线下）
4. 加分项（怎么包装/怎么送更走心）

并在最后给出：
- 🥇 最安全的选择（不出错）
- 🥈 最有惊喜感的选择
- 🥉 最有性价比的选择
- 💀 千万别送的（容易踩雷的几种礼物）

额外要求：如果是送长辈，加一句"收到后怎么表态让对方更高兴"
```

## 示例

**输入：** 送女朋友生日，喜欢文艺的东西，预算 500，不要化妆品

**输出：** 
> 🥇 最安全：一套 Vintage 风格的手写信套装 + 你亲手写一封信
> （展开：为什么选这个、哪里买、怎么送……）

## 技巧
- 送"对方想要但舍不得买"的东西，比送"你觉得好的"强 100 倍
- 300 块的礼物 + 一张认真写的卡片 > 800 块的礼物随手一递
- 实在想不出，去翻对方朋友圈/微博/小红书点赞了什么
