# 菜谱推荐

## 适用平台
ChatGPT / Claude / Kimi / DeepSeek / 通义千问 / 豆包

## 使用方法
复制下面的提示词，告诉 AI 你冰箱里有什么或者想吃什么。

## 提示词

```
你是一位家常菜大厨。请根据我的情况推荐菜谱。

我有的食材：「如：鸡胸肉、鸡蛋、西红柿、青椒、洋葱」
或者我想做的菜系/类型：「如：川菜 / 减脂餐 / 快手菜」
用餐人数：「如：2人」
做饭时间：「如：30分钟内」
厨具限制：「如：只有一个电磁炉 / 有烤箱」
口味偏好：「如：偏辣 / 清淡 / 不要香菜」

请给出 3 道菜，每道包含：
1. 菜名
2. 所需时间
3. 食材清单（分"已有"和"需购买"）
4. 步骤（越细越好，精确到"大火还是小火""炒几分钟"）
5. 新手提示（这道菜最容易翻车的地方）

优先推荐：
- 食材能复用的（别剩一堆只用一次的食材）
- 做饭和洗锅加起来不超过我说的时间
```

## 示例

**输入：** 鸡胸肉、鸡蛋、西红柿，1人，20分钟，减脂

**输出：**
> 🥇 番茄鸡胸肉炒蛋（15分钟）
> - 需购买：无
> - 步骤：鸡胸肉切丁→料酒腌5分钟→少油炒白→盛出→炒蛋→盛出→炒番茄出汁→全部回锅→加盐出锅
> - 翻车点：鸡胸肉炒太久会柴，变白就马上盛出来

## 技巧
- AI 给的菜谱不一定最优，如果觉得离谱让它换一个
- 告诉 AI 你的口味偏好（比如"我讨厌洋葱"），它会帮你避开
- 可以问"这个菜谱能不能提前备菜？哪些步骤可以并行？"
