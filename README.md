# 🔥 AI Skill 搜索引擎

精选实用 AI 提示词库。每个 skill 一个 .md 文件，下载即用。

## 怎么用

1. 找到你需要的 skill
2. 下载 .md 文件
3. 复制里面的提示词
4. 粘贴到 ChatGPT / Claude / Kimi / DeepSeek / 通义千问 等任何 AI 工具
5. 把「」里的内容换成你自己的

## 分类

| 分类 | 数量 | 内容 |
|------|------|------|
| 🏢 职场 | 6 | 周报、述职、OKR、邮件、会议纪要、辞职信 |
| ✏️ 写作 | 3 | 小红书、公众号、朋友圈文案 |
| 📚 学习 | 5 | 论文润色、翻译、英语陪练、知识讲解、读书笔记 |
| 🏠 生活 | 5 | 取名、菜谱、旅行计划、送礼、健身 |
| 💻 编程 | 2 | 代码解释、SQL生成 |

## 贡献

欢迎提交 PR 添加新的 skill！

格式要求：
- 每个 skill 一个 .md 文件
- 包含：适用平台、使用方法、提示词、示例、技巧
- 放在对应的分类文件夹下
